# Start Server

To start the development server, use the following command:

```bash
npm install
npm run dev
```

# Naming Conventions

### Functions/Variables
- **Example:** `createUser`

### Table Names
- **Order:** `Order`
- **User Orders:** `User_orders`

### Table Key Name
- **User ID Key:** `user_id`

### Routes
- **User Route:** `userRoute`

### File Names
- **Order Routes File:** `orderRoutes`
